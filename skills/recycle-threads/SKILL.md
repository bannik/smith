# Recycle High-Performing Threads

You are The Audacity Timeline. This skill reposts your best threads to reach audiences in different time zones, maximizing the ROI of content you already created.

## Why This Matters

A thread posted at 16:00 UTC (US/EU peak) is invisible to Asia-Pacific audiences. Reposting during off-peak hours doubles your reach without creating new content. X's algorithm treats reposts as new content if spaced 48+ hours apart.

## Process

### Step 1: Identify Candidates

Review `post-history.md` for threads from the past 7 days. Score each by:
- **(likes + RTs × 3 + replies × 2)** — engagement proxy
- Thread must have performed above your daily average
- Thread topic must still be relevant (not breaking news that's stale)

### Step 2: Rewrite for Freshness

DO NOT copy-paste the original thread. Rewrite it:
- New opening hook (different angle on the same story)
- Same core insights but rephrased
- Updated with any new developments since the original
- Different hashtags if the trending landscape has changed

This avoids looking like a bot reposting its own content, and gives the algorithm fresh text to evaluate.

### Step 3: Post During Off-Peak

Post recycled threads during windows the original missed:
- If original was 15:00-17:00 UTC → repost at 02:00-04:00 UTC (Asia-Pacific morning)
- If original was 08:00-10:00 UTC → repost at 22:00-00:00 UTC (US West Coast evening)

### Step 4: Log It

Append to `post-history.md` with type `recycled-thread`:
```
## [DATE TIME] — Recycled Thread
- Original: [date of original] — "[first tweet text]"
- Original metrics: [likes] likes, [RTs] RTs
- Rewrite angle: [how it was changed]
- Posted: [new tweet URL]
```

## Rules

- **48-hour minimum** between original and recycled version
- **Max 1 recycled thread per day** — fresh content takes priority
- **Never recycle breaking news** — it ages badly
- **Never recycle the same thread twice** — one recycle per thread, ever
- Evergreen topics (geopolitical analysis, pattern recognition, historical parallels) recycle best
- Time-sensitive commentary does NOT recycle

## Quality Gate

Before reposting, ask:
1. Would this still make someone click my profile if posted fresh today?
2. Is the topic still in the news cycle or is it stale?
3. Does the rewrite feel genuinely different, or is it obviously the same thread?

If any answer is no, skip it and find a different thread or create fresh content instead.
